# Отчёт по стандартизации

## Единая структура полей

Основные добавленные поля:

| Поле | Смысл |
|---|---|
| `feature_id` | стабильный ID объекта внутри тестового набора |
| `layer_id` | ID слоя |
| `source_layer` | исходное имя слоя |
| `category` | тип слоя: `admin`, `stats`, `settlements`, `buffer`, `special`, `points` |
| `time_label` | человекочитаемый период слоя |
| `year` | основной год объекта / среза |
| `start_year` | начало действия или датировки |
| `end_year` | конец действия или датировки |
| `name` | нормализованное имя объекта |
| `admin_parent` | нормализованная верхнеуровневая принадлежность |
| `unit_type` | нормализованный тип единицы |
| `population` | нормализованное население, если есть |
| `confidence` | нормализованная уверенность / качество |
| `uncertainty_flag` | 1, если объект содержит спорную / неясную / неустойчивую формулировку |
| `uncertainty_code` | машинный код статуса |
| `uncertainty_label` | человекочитаемый статус |
| `uncertainty_source_field` | поле, где найдена проблемная формулировка |
| `uncertainty_source_text` | исходный текст формулировки |
| `render_hatch` | 1, если объект надо штриховать |
| `render_outline` | `none`, если внешний контур отключён |

## Классы неопределённости

| Код | Подпись |
|---|---|
| `disputed_affiliation` | спорная принадлежность |
| `unclear_affiliation` | неясная принадлежность |
| `unstable_control` | неустойчивый контроль |
| `disputed_control` | спорный контроль |
| `unclear_boundary` | неясные границы |
| `transitional_zone` | переходная территория |
| `normal` | обычный объект |

## Индекс спорных объектов

Полный машинный индекс: `data/uncertainty_index.json`.
