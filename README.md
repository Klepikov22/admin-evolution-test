# Admin Evolution Rebuilt v3

Лёгкий статический просмотрщик для проверки GeoJSON-слоёв административной сетки.

## Что изменено в v3

- Все рабочие слои экспортированы как GeoJSON в единой системе координат: `EPSG:4326`.
- В Leaflet используется стандартная веб-проекция `EPSG:3857`; GeoJSON подаётся как долгота/широта.
- Растровая OSM-подложка полностью убрана.
- Подложка пересобрана как вектор:
  - `data/base/water.geojson` = Ocean + Lakes;
  - `data/base/rivers.geojson` = Rivers;
  - водохранилища исключены по `Reservoir / водохранилище`.
- Стартовый экстент взят по слою `018_Границы1798_Тоб_губ5_БРЭ.geojson` + примерно 200 км с запада и востока.
- Минимальный зум: `5`.
- Жёсткий `maxBounds` не применяется: он вызывал рывки при зуме на широких экранах. Ограничение реализовано через стартовый экстент, reset-view и клип данных.

## Запуск локально

```bash
python -m http.server 8000
```

Открыть:

```text
http://localhost:8000
```

## GitHub Pages

Залить содержимое папки в репозиторий, затем:

```text
Settings → Pages → Deploy from branch → main → /root
```
